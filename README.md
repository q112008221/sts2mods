## 📦 Mod 使用说明（玩家）

欢迎使用本 Mod！请按照以下步骤安装：

1. **下载 Release 包**  
   前往本仓库的 [Releases](https://github.com/q112008221/sts2mods/releases/) 页面，下载最新版本的压缩包（zip 或 rar 格式）。

2. **解压文件**  
   将下载的压缩包解压，得到 Mod 文件夹。

3. **放置到游戏目录**  
   将解压后的文件夹放入游戏目录下的 `mods` 文件夹。例如：

   `D:\SteamLibrary\steamapps\common\Slay the Spire 2\mods`


4. **启动游戏**  
打开《杀戮尖塔2》，在设置界面确认本 Mod 已启用即可。
